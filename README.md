# ACM@WUT MainPage & OnlineJudge业务系统


## 技术栈

### 服务端

- 语言：PHP
- 框架：ThinkPHP 5.1

### 前端

- 框架：Vue2
- UI：手撸
- 代码编辑器：CodeMirror
- 页面最小分辨率：1280*660 小于该尺寸Warning
